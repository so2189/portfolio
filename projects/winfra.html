    • Winfra

    A full-stack civic web platform designed to empower New Yorkers to report, discuss, and improve urban infrastructure in real time. Through an interactive map interface and community features, users can flag accessibility issues, propose solutions, and connect directly with local officials. This project was created with three teammembers as our senior design project at NYU Tandon, with a focus on Software Engineering principles, we collaborated on all aspects of this project in an Agile workflow.

    Overview

    Winfra is a civic tech platform designed to improve urban infrastructure through community reporting and collaboration. Users can submit real-time accessibility reports, browse an interactive map of issues, and advocate for solutions—especially addressing the needs of disabled New Yorkers. This project was developed using modern full-stack technologies and focuses on accessibility, community design, and local impact.

    Features

    • ✦ Real-time infrastructure issue reporting via an interactive city map
    • ✦ Visual tagging of reports in NYC and local map region
    • ✦ Account creation with personalized dashboards to view and manage user reports
    • ✦ Authentication and role-based access for citizens and officials
    • ✦ Community discussion threads on each report to drive public awareness
    • ✦ Mobile-friendly, accessibility-first UI for reporting on the go
    • ✦ Location sharing to surface live info about local infrastructure and reps
    • ✦ Automatic email drafting to local representatives based on user location
    • ✦ Commenting, sharing, and engagement tools to amplify community voice
