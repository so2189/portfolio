    • Fantasy Sports League Management System

    A full-stack web application that allows users to create and manage fantasy sports leagues. The project showcases end-to-end development—from database design to interface implementation with a strong focus on relational schema design, SQL integration, and user-centered design.

    Overview

    This project was designed to create a platform where users can build and manage their own fantasy sports leagues. It incorporates secure authentication, dynamic interfaces, and a relational database built from the ground up. Developed over the course of several weeks, it involved front-end design, SQL schema planning, and backend PHP development.

    Features

    • ✦ Create and manage custom fantasy sports leagues with real-time updates
    • ✦ Role-based access control: Developer vs. User, with distinct permissions
    • ✦ Secure login system with SHA2 password hashing and session-based authentication
    • ✦ Relational database schema built for scalability and integrity (users, leagues, teams, players)
    • ✦ Developer-only data export functionality based on account privileges
    • ✦ SQL-level privilege management using GRANT and REVOKE commands
    • ✦ User dashboard to edit profile info and manage teams/leagues
    • ✦ Redirect logic based on login state and privilege level
