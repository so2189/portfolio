    Overview

    Java Kahoot is a desktop-based trivia game that allows multiple players to connect and compete by answering questions in real-time. Built in Java, the project demonstrates the integration of thread concurrency, networking, and file I/O. The host runs the game and shares a PIN and IP address for others to join using their own devices. Questions are pulled from a customizable JSON file using the JSON Simple library, allowing for flexible game content. This project was created collaboratively for a final project for Java at NYU.

    Features

    • ✦ Multiplayer trivia game hosted on a local machine via IP-based socket networking
    • ✦ Players join using a game PIN and username through their own devices
    • ✦ Thread concurrency ensures accurate detection of who answers each question first
    • ✦ File I/O handles reading trivia question sets from JSON files for dynamic gameplay
    • ✦ End-of-game scoreboard ranks players based on speed and accuracy
    • ✦ Easily customizable question sets using JSON Simple library
    • ✦ Main gameplay visible on host screen, with real-time updates pushed to player clients
