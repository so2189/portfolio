    • Tic-Tac-Toe (R)

    A terminal-based Tic-Tac-Toe game built in R. Features interactive prompts, game visualization via base R plots, and full game loop logic with user validation and win checking.

    Visit GitHub

    Overview

    This project is a console-based 2-player game of Tic-Tac-Toe written in R. The game logic checks user input, updates the board, switches players, and visually renders the board in each round using R’s plotting system.

    How to Play

    • Run the function game() in your R console.
    • Players are prompted to input a row and column (1–3) on their turn.
    • Invalid moves are rejected and re-asked until valid.
    • The board is drawn after each move using base R plotting.
    • The game ends with a winner (3 in a row) or a stalemate (full board).

    Features

    • Interactive terminal prompts for each move.
    • Turn-based control with symbol assignment (Player 1 = X, Player 2 = O).
    • Victory detection across rows, columns, and diagonals.
    • Plot-based visual board that updates in real-time.

    Design Choices

    The game logic is modularized into functions: board creation, move validation, state checking, and UI rendering. Each part is testable and written for clarity and teaching-friendly use.
