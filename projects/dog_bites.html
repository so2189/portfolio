    • Dog Bites Visualization

    An interactive Unity visualization of NYC dog bite reports, letting players explore the data in 3D from a player perspective, with dogs representing bite incidents per zip code.

    Overview

    This project visualizes NYC dog bite data using Unity, allowing players to walk through and explore a 3D representation of bite incidents. Each dog prefab represents a reported bite, organized by zip code in Brooklyn for the year 2020. The visualization provides an immersive and interactive way to understand the distribution of incidents across the city.

    Process & Features

    • ✦ Used NYC Open Data DOHM Dog Bite dataset (Brooklyn, 2020, 551 reports)
    • ✦ Created 3D dog prefabs to represent bite reports per zip code
    • ✦ Implemented a player-controlled walkthrough with movement and jumping
    • ✦ Organized huddles of dog prefabs on a grid without overlap
    • ✦ Randomized rotation and positioning of dog objects within huddles
    • ✦ Interactive data visualization designed for exploratory learning
    • ✦ Future improvements: trigger animations, enhanced zip code visualization, Angry/Walking dog animations
