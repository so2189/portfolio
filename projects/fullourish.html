    Fullourish is a Give Back Project created as part of the Girls Who Code Leadership Academy in partnership with RTX. It is a web platform that helps people following plant-based diets find recipes tailored to their health goals and dietary needs. The project reflects the program’s core values by addressing a real-world challenge with an inclusive, user-focused solution that makes nutrition more accessible.

    Visit Site

    Overview

    This project is a website designed to support individuals following plant-based diets by helping them discover nutritionally fulfilling recipes tailored to their personal health needs. Using the Spoonacular API, the platform retrieves a wide variety of recipes and provides insight into the nutritional impact of each meal, empowering users to make informed food choices. The tool is especially helpful for those with dietary restrictions or specific health goals, offering an inclusive and customizable experience.

    Features

    • ✦ Filter recipes by dietary needs, health conditions, and nutritional goals
    • ✦ Calculate nutritional benefits of each recipe based on user inputs
    • ✦ Personalized suggestions based on dietary profiles and preferences
    • ✦ Track daily nutrient intake, such as how much more protein is needed
    • ✦ Save favorite recipes to a personalized dashboard
    • ✦ Accessible and intuitive user interface with mobile-friendly design
    • ✦ Uses Spoonacular API to pull accurate and diverse recipe data
    • ✦ Progress tracking to support long-term dietary goals
    • ✦ Designed to be inclusive for users managing chronic health issues
